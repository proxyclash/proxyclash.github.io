  1. 打开V2RayNG，点击底部导航栏的"订阅"选项。

  1. 返回主界面，您将看到所有可用的节点列表。
  2. 点击节点右侧的"ping"按钮测试延迟，选择延迟最低的节点。

常见问题解决方案

连接失败怎么办？

  1. 检查订阅是否过期，尝试更新订阅。
  2. 切换不同协议类型（如从VMess改为VLESS）。
  3. 尝试不同的传输方式（如TCP改为WebSocket）。
  4. 检查本地网络是否正常，尝试切换WiFi/移动数据。

速度慢如何优化？

  1. 选择地理位置较近的节点（如日本、新加坡节点对中国用户通常较快）。
  2. 关闭不必要的加密选项以降低CPU开销。
  3. 在夜间等非高峰时段使用，避开网络拥堵。
  4. 考虑使用付费节点获取更好的带宽保障。

节点频繁掉线怎么处理？

  1. 启用"自动切换节点"功能，设置合理的切换阈值。
  2. 在"设置"中调整心跳间隔和超时时间。
  3. 使用支持TCP BBR加速的节点改善长距离连接稳定性。

FAQ

SSR 的端口复用有什么风险？
SSR 支持多端口配置以复用服务，但若端口过多或设置冲突，可能被扫描工具识别或误判为异常服务。建议保持配置简洁，并合理分配端口范围以减少暴露风险。
PassWall2 是什么软件？
PassWall2 是一个运行在 OpenWRT 路由器上的插件，用于代理流量分流和节点管理。它支持多种协议，包括 V2Ray、Trojan、SSR 等。通过 Web 界面，用户可以灵活设置不同设备走不同代理，适合家庭或小型局域网使用。
VLESS + Reality 模式如何隐藏流量特征？
Reality 模式通过伪装握手和真实网站特征，使流量难以被 DPI 检测。结合策略组使用，可有效隐藏节点通信特征，同时保持安全和高速传输，适合高封锁环境。
PassWall2 如何自动切换最佳节点？
通过延迟测速和策略组设置，PassWall2 可自动选择延迟最低的节点处理流量。用户无需手动切换节点，保证网页、视频和游戏流量的速度和稳定性，提高整体网络体验。
Netch 如何实现 UDP 游戏加速？
Netch 支持 UDP 转发，可专门为游戏流量建立高速通道。结合策略组和节点延迟检测，可有效减少丢包和延迟，提高游戏体验的流畅性和稳定性。
SSR 的 obfs 混淆原理是什么？
SSR 的 obfs 模块通过模拟常见流量特征（如 HTTPS 或 HTTP 请求头）来掩盖加密流量。这样可让代理流量看起来像正常的网页访问，从而有效降低被检测和封锁的风险。
在 Clash 中使用混合代理模式有什么好处？
Clash 的混合代理模式允许同时运行多个代理协议和端口监听，从而实现流量分流。它可以为不同的应用或域名设置不同的代理方式，提高灵活性与连接稳定性，非常适合多协议用户使用。
V2Ray 使用 WebSocket + TLS 有哪些优势？
WebSocket + TLS 可将 V2Ray 流量伪装成 HTTPS 请求，隐藏真实通信特征。结合域名伪装和策略组使用，可以有效绕过封锁，提高节点安全性和连接稳定性，保持长期稳定访问。
Clash 如何实现域名白名单策略？
Clash 可以通过策略组结合规则文件，将指定域名或网站加入白名单，实现直连而不经过代理节点。这样可以保证国内访问速度，同时节省节点带宽资源，提高整体网络效率。
VLESS + XTLS 在移动网络下的优势有哪些？
XTLS 模式减少握手次数和加密开销，适合高延迟或不稳定的移动网络。结合策略组和多节点切换，可提升网页浏览、视频播放和游戏的连接稳定性和流畅度。
